# 📘 Глоссарий проекта Travel World

Этот документ содержит основные термины, используемые в проекте.

---

## 🌍 SPA (Single Page Application)

Приложение, которое работает внутри одной HTML-страницы без полной перезагрузки при переходах между разделами.

---

## 🧭 Router

Модуль, отвечающий за навигацию между страницами приложения без перезагрузки браузера.  
Использует History API (`pushState`, `popstate`).

---

## 📄 Page

Абстракция страницы приложения.

Содержит:

- HTML разметку
- функцию жизненного цикла (mount)
- функцию очистки (unmount)

---

## 🔁 Mount

Процесс инициализации страницы:

- вставка HTML
- навешивание событий
- запуск логики

---

## 🧹 Unmount

Процесс очистки страницы:

- удаление обработчиков событий
- освобождение ресурсов

---

## 🔗 Navigate

Функция перехода между страницами без перезагрузки браузера.

---

## 🎫 Credentials

Объект авторизации пользователя, сохраняемый в `localStorage`.

Пример:

```js
{
  keyword: "TRAVEL2026",
  coupon: "TW-45821"
}
```

---

## 🎨 CSS Isolation

Подход, при котором стили страницы подключаются и удаляются динамически при переходах между страницами, предотвращая конфликты CSS.

---

## 🧠 Route Guard (будущее расширение)

Механизм проверки доступа к маршрутам (например, запрет доступа к `/offers` без авторизации).
