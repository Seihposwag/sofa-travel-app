## Спецификация бизнес-прецедентов

### BUC-01: Просмотр главной страницы

| Поле | Значение |
|---|---|
| Актор | Гость, Авторизованный пользователь |
| Цель | Получить обзор туристических направлений |
| Триггер | Переход на / или /home |
| Результат | Отображены карточки Токио, Парижа, Бали |
| Ограничения | Нет (доступно без авторизации) |

### BUC-02: Изучение направлений путешествий

| Поле | Значение |
|---|---|
| Актор | Гость, Авторизованный пользователь |
| Цель | Ознакомиться с популярными направлениями |
| Триггер | Скролл главной страницы, клик на карточку |
| Результат | Пользователь видит описание направления |
| Ограничения | Нет |

### BUC-03: Изучение информации о компании

| Поле | Значение |
|---|---|
| Актор | Гость, Авторизованный пользователь |
| Цель | Узнать о деятельности Travel World |
| Триггер | Клик «О нас» в навигации → /about |
| Результат | Загружена страница /about |
| Ограничения | Нет |

### BUC-04: Авторизация по купону

| Поле | Значение |
|---|---|
| Актор | Гость |
| Цель | Получить доступ к предложениям |
| Триггер | Клик «Ввести купон» → /auth |
| Основной поток | Ввод ключевого слова → ввод купона → подтверждение |
| Результат | Сессия сохранена в localStorage → редирект на /offers |
| Альт. поток | Неверный купон → сообщение об ошибке |

### BUC-05: Просмотр туристических предложений

| Поле | Значение |
|---|---|
| Актор | Авторизованный пользователь |
| Цель | Ознакомиться со специальными предложениями и турами |
| Триггер | Клик «Путевки» или переход на /offers |
| Результат | Отображён список туристических предложений |
| Ограничения | Требуется авторизация; гость → /auth |

### BUC-06: Навигация между разделами

| Поле | Значение |
|---|---|
| Актор | Гость, Авторизованный пользователь |
| Цель | Перемещение между страницами без перезагрузки |
| Триггер | Клик на пункт меню |
| Результат | Смена контента без HTTP-запроса к серверу |
| Механизм | History API, кастомный Router |

### BUC-07: Выход из системы

| Поле | Значение |
|---|---|
| Актор | Авторизованный пользователь |
| Цель | Завершить сессию |
| Триггер | Пользователь очищает localStorage или перезагружает |
| Результат | Сессия удалена, доступ к /offers закрыт |
