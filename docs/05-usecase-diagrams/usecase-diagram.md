## Спецификация вариантов использования

### UC1 — Просмотр списка направлений

| Атрибут | Значение |
|---|---|
| Акторы | Гость, Авторизованный пользователь |
| Предусловие | Приложение загружено в браузере |
| Основной поток | 1. Пользователь открывает / или /home → 2. Router монтирует homePage → 3. Отображаются карточки направлений (Токио, Париж, Бали) |
| Результат | Пользователь видит 3 карточки направлений с фото и описанием |
| Постусловие | History state = /home |

### UC2 — Просмотр страницы «О нас»

| Атрибут | Значение |
|---|---|
| Акторы | Гость, Авторизованный пользователь |
| Предусловие | Приложение загружено |
| Основной поток | 1. Клик «О нас» → 2. navigate('/about') → 3. Router unmount предыдущей, mount aboutPage → 4. CSS about.css подключается |
| Результат | Страница /about отображена, стили home.css удалены |

### UC3 — Авторизация по купону

| Атрибут | Значение |
|---|---|
| Акторы | Гость |
| Предусловие | Пользователь не авторизован |
| Основной поток | 1. Клик «Ввести купон» → 2. /auth монтируется → 3. Ввод ключевого слова + купона → 4. Сохранение в localStorage → 5. navigate('/offers') |
| Альтернативный | 3a. Некорректный формат → сообщение об ошибке, поля очищаются |
| Результат | credentials = {keyword, coupon, authorizedAt} в localStorage |

### UC4 — SPA-навигация без перезагрузки

| Атрибут | Значение |
|---|---|
| Акторы | Гость, Авторизованный пользователь |
| Предусловие | — |
| Основной поток | 1. Клик на ссылку → 2. preventDefault() → 3. navigate(path) → 4. history.pushState → 5. Router.render() → 6. unmount старой + mount новой страницы |
| Результат | Страница изменена, HTTP-запроса не было, URL обновлён |

### UC5 — Просмотр специальных предложений

| Атрибут | Значение |
|---|---|
| Акторы | Авторизованный пользователь |
| Предусловие | Авторизация выполнена (localStorage содержит credentials) |
| Основной поток | 1. Переход на /offers → 2. Router проверяет guard → 3. credentials валидны → 4. offersPage монтируется |
| Альтернативный | 2a. credentials отсутствуют → navigate('/auth') |
| Результат | Список туристических предложений отображён |

### UC6 — Выход из системы

| Атрибут | Значение |
|---|---|
| Акторы | Авторизованный пользователь |
| Предусловие | Авторизация активна |
| Основной поток | 1. localStorage.removeItem('credentials') → 2. navigate('/auth') |
| Результат | Сессия очищена, доступ к /offers закрыт |
