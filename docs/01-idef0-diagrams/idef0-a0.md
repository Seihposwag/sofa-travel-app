# IDEF0. Диаграмма A-0 — Контекстная диаграмма
> Проект: Travel World SPA  
> Вариант 8 — Блог о путешествиях

---

## Диаграмма контекста (A-0)

Диаграмма A-0 представляет систему как единый функциональный блок «Управление информацией о путешествиях».

```
           УПРАВЛЕНИЕ (Control)
    ┌────────────────────────────────────────────────────┐
    │  Бизнес-правила авторизации (купон + ключевое слово)│
    │  Правила маршрутизации (route guard /offers)        │
    │  CSS-политика изоляции стилей                      │
    └────────────────────────────────────────────────────┘
                          │
                          ▼
 INPUT                ┌───────────────────────────────┐               OUTPUT
                      │                               │
 Запрос пользователя  │   Управление информацией о    │  Отрендеренная HTML-страница
 (URL / клик) ───────▶│       путешествиях            │──────────────────────────────▶
                      │                               │
 Данные купона        │        (Travel World SPA)     │  Навигационное состояние
 (keyword + coupon) ──▶                               │  (History API state) ─────────▶
                      │                               │
 Статические данные   │                               │  Авторизованный/гостевой доступ
 о турах / странах ───▶                               │──────────────────────────────▶
                      └───────────────────────────────┘
                          ▲
                          │
          МЕХАНИЗМ (Mechanism)
    ┌────────────────────────────────────────────────────┐
    │  Браузер (Chrome / Firefox / Safari)               │
    │  History API (pushState / popstate)                │
    │  LocalStorage (хранение купона)                    │
    │  ES Modules (Vanilla JavaScript)                   │
    └────────────────────────────────────────────────────┘
```

---

## Расшифровка элементов

| Элемент | Тип | Описание |
|---|---|---|
| Запрос пользователя | Input | URL-переход или клик на ссылку навигации |
| Данные купона | Input | Ключевое слово + номер купона, вводимые на /auth |
| Статические данные о турах | Input | Встроенный контент: направления, предложения, описания |
| Бизнес-правила авторизации | Control | Купон обязателен для доступа к /offers |
| Правила маршрутизации | Control | Route guard: неавторизованный → /auth |
| CSS-политика изоляции | Control | Стили подключаются/удаляются при смене страницы |
| Браузер | Mechanism | Среда выполнения приложения |
| History API | Mechanism | Навигация без перезагрузки |
| LocalStorage | Mechanism | Персистентное хранение авторизации |
| ES Modules | Mechanism | Система модулей JavaScript |
| Отрендеренная HTML-страница | Output | Визуальный результат навигации |
| Навигационное состояние | Output | Текущий маршрут в History |
| Авторизованный/гостевой доступ | Output | Статус сессии пользователя |

## A-1. Декомпозиция: основные функции системы

```
┌─────────────────────────────────────────────────────────────────┐
│                    Travel World SPA (A-0)                       │
│                                                                 │
│  ┌──────────────┐   ┌──────────────┐   ┌──────────────────┐   │
│  │   A1         │   │   A2         │   │   A3             │   │
│  │  Маршрути-   │──▶│  Управление  │──▶│  Рендеринг       │   │
│  │  зация       │   │  авторизацией│   │  страниц         │   │
│  └──────────────┘   └──────────────┘   └──────────────────┘   │
│         │                                       │               │
│         ▼                                       ▼               │
│  ┌──────────────┐                    ┌──────────────────┐      │
│  │   A4         │                    │   A5             │      │
│  │  Управление  │                    │  Изоляция CSS    │      │
│  │  состоянием  │                    │  по страницам    │      │
│  └──────────────┘                    └──────────────────┘      │
└─────────────────────────────────────────────────────────────────┘
```

| Блок | Назначение |
|---|---|
| A1 — Маршрутизация | Обработка URL, регистрация маршрутов, route guard |
| A2 — Авторизация | Проверка купона, сохранение в localStorage |
| A3 — Рендеринг страниц | Mount/unmount жизненного цикла страниц |
| A4 — Управление состоянием | История переходов, текущая страница |
| A5 — Изоляция CSS | Динамическое подключение/удаление стилей |
