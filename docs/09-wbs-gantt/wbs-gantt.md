## Управление рисками

| № | Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|---|
| R1 | Несовместимость History API в старых браузерах | Низкая | Среднее | Протестировать в Chrome, Firefox, Safari |
| R2 | localStorage заблокирован (режим инкогнито) | Средняя | Высокое | Обработать ошибку записи, показать предупреждение |
| R3 | Конфликт CSS при быстром переключении страниц | Низкая | Низкое | CSSManager гарантирует удаление предыдущего link |
| R4 | Прямой доступ по URL без роутера (404) | Средняя | Среднее | Настроить dev-сервер (Vite) на fallback index.html |
| R5 | Срыв срока сдачи (14 мая) | Низкая | Высокое | Буфер 1 неделя; приоритет: реализация → документация |
