# 📙 Руководство пользователя — Travel World

---

## 🚀 Описание приложения

Travel World — это демонстрационное SPA-приложение туристического агентства.

Приложение позволяет:

- просматривать направления путешествий
- изучать информацию о компании
- смотреть специальные предложения
- проходить авторизацию по купону

---

## 🧭 Навигация

В приложении доступно несколько страниц:

- `/home` — главная страница
- `/about` — информация о компании
- `/offers` — туристические предложения
- `/auth` — авторизация

Переходы между страницами происходят без перезагрузки сайта.

---

## 🔐 Авторизация

Для доступа к предложениям необходимо:

1. Перейти на страницу авторизации
2. Ввести:
   - ключевое слово
   - номер купона
3. Нажать кнопку "Войти"

После успешной авторизации данные сохраняются в браузере.

---

## 🎫 После авторизации

Пользователь получает доступ к странице:

```

/offers

```

и может просматривать специальные предложения.

---

## ⚠️ Важно

- Приложение работает без сервера
- Все данные хранятся в браузере (localStorage)
- Перезагрузка страницы не сбрасывает авторизацию

---

## 🧠 Особенности

- нет перезагрузки страниц
- быстрая навигация
- изолированные стили страниц
- клиентская логика без backend
