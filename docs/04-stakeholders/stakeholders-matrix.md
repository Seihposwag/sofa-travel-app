# Матрица стейкхолдеров — Travel World SPA
> Проект: Travel World SPA | Вариант 8 — Блог о путешествиях

---

## Матрица заинтересованных лиц

| № | Стейкхолдер | Роль | Интерес | Влияние | Стратегия |
|---|---|---|---|---|---|
| 1 | Конечный пользователь (турист) | Посетитель сайта | Получить информацию о турах, найти предложение | Высокое | Приоритетная аудитория; UX и контент направлены на него |
| 2 | Менеджер турагентства | Заказчик | Привлечение клиентов, демонстрация предложений | Высокое | Обеспечить витрину туров и легко обновляемый контент |
| 3 | Разработчик (студент) | Исполнитель | Изучение SPA-архитектуры, академическая оценка | Высокое | Чёткие требования к коду и документации |
| 4 | Преподаватель (Самойлов Ф.В.) | Принимающий | Соответствие требованиям курсового проекта | Высокое | Строгое следование методическим указаниям |
| 5 | Гость (потенциальный клиент) | Посетитель | Быстрый просмотр направлений без регистрации | Среднее | Открытый доступ к основному контенту |
| 6 | Авторизованный клиент | Клиент с купоном | Просмотр эксклюзивных предложений | Среднее | Плавная авторизация, понятный UX |
| 7 | Браузер / платформа | Техническая среда | Стандарты Web API, безопасность | Низкое | Использование стандартных Web API без vendor lock-in |

```
  │                              Высокий

```

| Квадрант | Стейкхолдеры | Стратегия |
|---|---|---|
| Высокое влияние / Высокий интерес | Конечный пользователь, Менеджер, Преподаватель, Разработчик | Активное вовлечение, регулярная коммуникация |
| Высокое влияние / Низкий интерес | — | — |
| Низкое влияние / Высокий интерес | Гость, Авторизованный клиент | Информировать, обеспечить хороший UX |
| Низкое влияние / Низкий интерес | Браузер / платформа | Мониторинг совместимости |
